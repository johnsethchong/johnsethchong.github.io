Dollar Menu Money

Project conducted using Python on 100 random cities and counties to determine if there is a direct correlation between areas with lower household incomes and the amount of fast food restaurants. Extracted census and API Yelp data to create DataFrames, various charts, and a heat map for analysis. Concluded that there is no significant correlation between income and fast food restaurants in cities, but evidence suggests there is a correlation with counties.
